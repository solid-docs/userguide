# SolidOS User Guide

User documentation for the SolidOS Databrowser - learn to manage your data, connect with contacts, and control access to your Solid Pod.

**Live site:** https://solid-docs.github.io/userguide/

## Development

This site is built with [Docusaurus 3](https://docusaurus.io/).

### Prerequisites

- Node.js 20+

### Local Development

```bash
npm install
npm start
```

This starts a local development server at http://localhost:3000/userguide/

### Build

```bash
npm run build
```

Generates static content into the `build` directory.

## Contributing

Contributions are welcome! Please feel free to submit issues and pull requests.

Documentation is in the `docs/` directory:
- `docs/getting-started/` - Introduction, login, navigation, creating data
- `docs/views/` - Documentation for each SolidOS view/pane
- `docs/reference/` - User roles, preferences, FAQ
