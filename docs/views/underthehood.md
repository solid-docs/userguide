# Under the Hood View

The Under the Hood view displays information about the resource that may be of interest, including the URL to the resource and how it could be directly edited. Additionally, you can delete the resource if you have permission to do so.

![Delete an existing Resource](/img/views/underthehood/Delete_Resource.png)

## Prerequisite

To have the privileges to manipulate your own resources you have to:

1. Login
2. Click on your profile image. A menu unfolds.
3. Click on **Preferences**.
4. Select **Power user** **and** **Developer** under **User types**.

See [User Roles](/reference/user-roles) for more information about user types.

## Deleting a Resource

You can delete existing folder and data resources to which you have delete permission.

To delete an existing data resource:

1. Navigate to and open the resource you want to delete.
2. Click on the ![Under the Hood](https://solidos.github.io/solid-ui/src/originalIcons/tango/22-emblem-system.png) Under the Hood icon.
3. In the displayed panel, hover your mouse cursor to the left of the ![Refresh](https://solidos.github.io/solid-ui/src/icons/noun_479395.svg) Refresh icon. A ![Remove](https://solidos.github.io/solid-ui/src/icons/noun_2188_red.svg) Remove icon will become visible.
4. Click the Remove icon to delete the resource.
5. Confirm the deletion by clicking the **Delete file** button.

:::tip
You need to refresh your web browser for the deletion to be visible in the Folder view.
:::
